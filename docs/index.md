# ForgeText Docs

This folder backs the public GitHub Pages site and the in-repository product docs.

ForgeText is currently documented as a Studio-first native editor with:

- raw-first editing defaults for structured files
- `Workbench + Advanced` runtime, privacy, and safety controls
- managed enterprise policy support
- local-first AI provider setup with cloud and local-model flows
- Sparkle/GitHub Release update guidance

- [Help](./HELP.md): ForgeText 1.2.1 feature guide, trust model, and common workflows
- [Managed Policy](./MANAGED_POLICY.md): enterprise admin controls, file locations, and sample policy
- [Updates](./UPDATES.md): Sparkle, appcast, DMG, and GitHub Releases setup
- [Roadmap](./ROADMAP.md): shipped scope and future priorities
- [UI Workbench Plan](./UI_WORKBENCH_PLAN.md): Studio-first workbench direction and UI principles
- [Appcast](./appcast.xml): public Sparkle update feed
